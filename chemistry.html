上

九年级上册

一走进化学世界 九上▼
  • 物理变化 vs 化学变化：是否有新物质生成。化学变化常伴随发光、放热、颜色变化、产生气体或沉淀，但不能以现象为判断依据。
  • 物理性质：颜色、状态、气味、熔点、沸点、硬度、密度、溶解性、导电性等（不需化学变化就表现）。
  • 化学性质：可燃性、助燃性、氧化性、还原性、酸碱性、稳定性等（在化学变化中表现）。
  • 基本操作：药品取用"三不"原则（不触、不闻、不尝）。酒精灯用外焰加热，熄灭用灯帽盖灭（不可吹灭）。
发光放热不一定是化学变化！如灯泡发光是物理变化。判断唯一标准：是否有新物质生成。
二我们周围的空气 九上▼
  • 空气成分：N₂ 78%、O₂ 21%、稀有气体 0.94%、CO₂ 0.03%。测定氧气含量的实验（红磷燃烧）原理：消耗 O₂ 使瓶内气压减小。
  • O₂ 性质：无色无味、密度略大于空气、不易溶于水。支持燃烧（助燃性），化学性质活泼。检验：带火星木条复燃。
实验室制 O₂ 三法：
2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑（固固加热，试管口塞棉花）
2H₂O₂ →MnO₂ 2H₂O + O₂↑（固液不加热，MnO₂ 是催化剂）
  • 催化剂：改变反应速率，本身的质量和化学性质在反应前后不变（"一变两不变"）。
三物质构成的奥秘 九上▼
原子结构：质子数 = 核外电子数 = 核电荷数 = 原子序数
相对原子质量 ≈ 质子数 + 中子数
  • 元素：质子数（核电荷数）相同的一类原子的总称。地壳中含量前四：O > Si > Al > Fe。
  • 化合价口诀：一价钾钠氯氢银，二价氧钙钡镁锌；三铝四硅五价磷，二三铁二四碳；铜汞二价最常见。
  • 化学式书写：正价在前负价在后，交叉约简。
元素是宏观概念，只讲种类不讲个数。不能说"两个氧元素"，只能说"氧元素"或"两个氧原子"。
四自然界的水 九上▼
  • 电解水：正极产生 O₂，负极产生 H₂，体积比 1:2。结论：水由氢元素和氧元素组成。
  • 水的净化：沉淀 → 过滤 → 吸附（活性炭）→ 消毒。其中过滤除去不溶性杂质，吸附除去色素和异味。
  • 硬水 vs 软水：含较多可溶性钙镁化合物的水是硬水。用肥皂水鉴别（浮渣多为硬水），煮沸可软化硬水。
五化学方程式 九上▼
质量守恒定律：参加化学反应的各物质的质量总和 = 反应后生成的各物质的质量总和。微观解释——化学反应前后原子种类、数目、质量不变。
  • 化学方程式书写：① 写（反应物和生成物的化学式）② 配（配平，使两边各原子个数相等）③ 标（注明反应条件和 ↑↓）。
配平只用最小公倍数法或奇偶数法，不能改变化学式中的角标数字！配平是改系数，不是改化学式。
六碳和碳的氧化物 九上▼
  • 碳单质：金刚石（最硬）、石墨（导电、滑腻）、C₆₀。同种元素可组成不同单质（同素异形体）。
  • CO：无色无味、有毒（与血红蛋白结合）、可燃（2CO+O₂→2CO₂）、还原性（炼铁）。
  • CO₂：无色无味、密度大于空气、不支持燃烧（灭火）。检验：通入澄清石灰水变浑浊（CO₂+Ca(OH)₂→CaCO₃↓+H₂O）。
实验室制 CO₂：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑（不能用稀硫酸，生成微溶 CaSO₄ 包裹石灰石阻止反应）。
CO₂ 不能支持燃烧 ≠ CO₂ 不能燃烧。CO₂ 本身不燃烧也不支持燃烧，但不是所有物质都不能在 CO₂ 中燃烧（如镁条可以在 CO₂ 中燃烧：2Mg+CO₂→2MgO+C）。
七燃料及其利用 九上▼
燃烧三条件（缺一不可）：① 可燃物 ② 与 O₂ 接触 ③ 温度达到着火点。
灭火原理（破坏任一条件）：① 清除可燃物 ② 隔绝 O₂ ③ 降温至着火点以下。
  • 三大化石燃料：煤、石油、天然气（CH₄，燃烧：CH₄+2O₂→CO₂+2H₂O）。都是不可再生能源。

九年级下册

八金属和金属材料 九下▼
金属活动性顺序（由强到弱）：
K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au
规律：① 排在 H 前的金属能置换出酸中的氢 ② 排在前面的金属能把后面的金属从盐溶液中置换出来（K、Ca、Na 除外，它们先与水反应）。
  • 合金：金属与金属（或非金属）熔合而成，具有金属特性。合金硬度一般大于纯金属，熔点一般低于纯金属。常见合金：生铁、钢、黄铜、铝合金。
  • 炼铁：Fe₂O₃ + 3CO →高温 2Fe + 3CO₂（还原反应，CO 是还原剂）。
  • 铁锈：主要成分 Fe₂O₃·xH₂O。防锈措施：保持干燥、涂油、刷漆、镀金属、制成不锈钢。
铁与酸/盐溶液发生置换反应生成的是 +2 价亚铁盐（Fe²⁺，溶液浅绿色），不是 +3 价铁盐！如 Fe + 2HCl → FeCl₂ + H₂↑（不是 FeCl₃）。
九溶液 九下▼
  • 溶液：均一、稳定的混合物。溶质+溶剂→溶液。水是最常见的溶剂。
  • 溶解度：一定温度下，100g 溶剂中最多溶解的溶质质量（g）。大多数固体溶解度随温度升高而增大（KNO₃），少数变化不大（NaCl），极少数随温度升高而减小（Ca(OH)₂）。
溶质质量分数：ω = m溶质 / m溶液 × 100%（m溶液 = m溶质 + m溶剂）
稀释公式：浓溶液质量 × 浓浓度 = 稀溶液质量 × 稀浓度
十酸和碱 九下▼
  • 常见酸：HCl（盐酸）、H₂SO₄（硫酸）。酸的通性：与指示剂（石蕊变红）、与活泼金属（生成 H₂）、与金属氧化物、与碱（中和）、与某些盐。
  • 常见碱：NaOH（烧碱/火碱/苛性钠，易潮解）、Ca(OH)₂（熟石灰/消石灰）。碱的通性：与指示剂（石蕊变蓝、酚酞变红）、与非金属氧化物、与酸（中和）、与某些盐。
中和反应：酸 + 碱 → 盐 + 水（HCl + NaOH → NaCl + H₂O）。实质：H⁺ + OH⁻ → H₂O。
pH：pH < 7 酸性，pH = 7 中性，pH > 7 碱性。pH 试纸测的是整数。
十一盐 化肥 九下▼
复分解反应条件：生成物中有↓（沉淀）、↑（气体）或 H₂O。至少满足一个才能发生。
常见沉淀：CaCO₃↓（白）、BaSO₄↓（白，不溶于酸）、AgCl↓（白，不溶于酸）、Cu(OH)₂↓（蓝）、Fe(OH)₃↓（红褐）。
  • 化肥：氮肥（促叶）、磷肥（促根/果实）、钾肥（促茎秆，抗倒伏）。复合肥：KNO₃、NH₄H₂PO₄。
复分解反应不一定都能发生！必须满足"生成↓/↑/H₂O"之一。如 NaCl + KNO₃ → 不发生反应（没有↓/↑/H₂O生成）。
十二化学与生活 九下▼
  • 六大营养素：蛋白质、糖类、油脂、维生素、无机盐、水。其中供能的是蛋白质、糖类、油脂。
  • 有机合成材料：塑料、合成纤维、合成橡胶。天然有机材料：棉花、羊毛、天然橡胶、蚕丝。

化学易错点 Top 10

  1. 物理变化 vs 化学变化——唯一判断标准是有无新物质生成，发光/放热/变色只是辅助现象，不能作为判断依据。
  2. 催化剂不参与反应？——催化剂参与反应，只是在反应后质量和化学性质不变。"一变两不变"：改变反应速率，质量和化学性质不变。
  3. 铁与酸/盐的置换——生成的是 +2 价亚铁盐（Fe²⁺，溶液浅绿色），不是 +3 价铁盐。FeCl₂ vs FeCl₃ 经常考。
  4. 元素是宏观概念——只讲种类不讲个数。"两个氧元素"是错误说法，应说"氧元素"或"两个氧原子"。
  5. 配平是改系数不改变化学式——绝对不能改动化学式中右下角的角标数字来配平！配平只能改前面的系数。
  6. CO₂ 不支持燃烧 ≠ 所有物质都不能在其中燃烧——如镁条可在 CO₂ 中燃烧：2Mg + CO₂ → 2MgO + C。
  7. 复分解反应不一定发生——必须满足"生成沉淀/气体/水"之一。如 NaCl + KNO₃ 不发生反应。
  8. 溶解度与溶质质量分数——溶解度是 100g 溶剂中的最大溶质质量（温度相关），溶质质量分数是溶质占溶液的比例。
  9. 中和反应 = 酸 + 碱 → 盐 + 水——但生成盐和水的反应不一定是中和反应！如 CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O 不是中和反应（CO₂ 不是酸）。
  10. 浓硫酸稀释——酸入水，沿壁倒，不断搅拌！绝对不能把水倒入浓硫酸中（会暴沸飞溅）。
