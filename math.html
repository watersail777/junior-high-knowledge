七

七年级

（上·下两册）
一有理数 七上▼
  • 正数与负数：大于 0 的数是正数，小于 0 的数是负数。0 既不是正数也不是负数。
  • 数轴三要素：原点、正方向、单位长度。
  • 相反数：只有符号不同的两个数互为相反数。a 的相反数是 −a。数轴上，相反数关于原点对称。
  • 绝对值：|a| 表示数 a 的点到原点的距离。|a| ≥ 0，|a| = |−a|。
  • 有理数分类：整数（正整数、0、负整数）+ 分数（正分数、负分数）。有限小数和无限循环小数都是分数。
数轴 · 绝对值是点到原点的距离
加法法则：同号相加取相同符号，绝对值相加；异号相加取绝对值大的符号，绝对值相减。
减法：减去一个数 = 加上它的相反数。 a − b = a + (−b)
乘法：同号得正，异号得负，绝对值相乘。几个数相乘，负因数有奇数个时积为负。
除法：除以一个不为 0 的数 = 乘以它的倒数。
负数的偶次幂是正数，奇次幂是负数。如 (−2)² = 4，但 −2² = −4（注意括号！）
  • 科学记数法：a × 10ⁿ，其中 1 ≤ |a| < 10。
二整式的加减 七上▼
  • 单项式：数或字母的积。如 3x²y（系数 3，次数 2+1=3）。
  • 多项式：几个单项式的和。次数最高项的次数就是多项式的次数。
  • 同类项：字母相同 + 相同字母的指数相同。合并时系数相加，字母部分不变。
  • 去括号：括号前是"+"不变号；括号前是"−"各项都变号。
去括号时最容易漏乘！如 2(x−3y) = 2x−6y，不要只乘第一项。另外 −(a−b) = −a+b，不要忘记括号内后面的项也要变号。
三一元一次方程 七上▼
解方程五步法：① 去分母 ② 去括号 ③ 移项（过等号要变号！）④ 合并同类项 ⑤ 系数化为 1
去分母时，不含分母的项也要乘！例如：x/2 + 1 = x/3 → 两边乘 6 得 3x + 6 = 2x，不是 3x + 1 = 2x。
典型例题
某数加上 5 后乘以 2 等于 18，求该数。
解：设该数为 x。2(x+5)=18 → x+5=9 → x=4。检验：2×(4+5)=18 ✓
四几何图形初步 七上▼
  • 直线、射线、线段：直线无端点，射线 1 个端点，线段 2 个端点（可度量）。
  • 两点之间线段最短（公理）。两点间线段的长度叫距离。
  • 线段中点：若 M 是 AB 中点，则 AM = MB = ½AB。
  • 角：1 周角 = 360°，1 平角 = 180°，1 直角 = 90°。1° = 60′，1′ = 60″。
  • 余角：和为 90°。同角（等角）的余角相等。
  • 补角：和为 180°。同角（等角）的补角相等。
五相交线与平行线 七下▼
三线八角 · 平行线的判定与性质
判定（由角推平行）：同位角相等 / 内错角相等 / 同旁内角互补 → 两直线平行
性质（由平行推角）：两直线平行 → 同位角相等 / 内错角相等 / 同旁内角互补
判定和性质容易搞混！判定是"由角的关系推出线平行"；性质是"已知线平行推出角的关系"。
六实数 七下▼
  • 平方根：若 x² = a，则 x 是 a 的平方根。正数有 2 个平方根（互为相反数），负数无平方根。
  • 算术平方根：正数 a 的正平方根 √a。√a ≥ 0（非负性）。
  • 立方根：任何数都有唯一的立方根 ³√a。
  • 无理数：无限不循环小数（π、√2、√3…）。实数 = 有理数 + 无理数。
√16 = 4（不是 ±4！）算术平方根只有一个非负值。但解 x² = 16 时 x = ±4，两个都是平方根。
七平面直角坐标系 七下▼
四个象限 · 点的坐标 (x, y)
  • x 轴上的点纵坐标为 0；y 轴上的点横坐标为 0。
  • 对称：关于 x 轴对称，横坐标相同纵坐标互为相反数；关于 y 轴对称反之。
八二元一次方程组 七下▼
代入消元法：从某个方程解出一个未知数，代入另一方程。
加减消元法：使同一未知数的系数相等或互为相反数，两式相加或相减消元。
典型例题
解 { x + y = 10, 2x − y = 5 }
加减法：两式相加 → 3x = 15 → x = 5 → y = 5。
九不等式与不等式组 七下▼
不等式解集在数轴上的表示：≥、≤ 用实心；>、< 用空心
不等式基本性质：① 两边同加（减）同一个数，不等号方向不变。② 两边同乘（除）同一个正数，不等号方向不变。③ 两边同乘（除）同一个负数，不等号方向要改变！
乘除负数忘记改变不等号方向，是绝对的高频错误！如 −2x > 6 → x < −3（不等号由 > 变为 <）。
  • 不等式组口诀：同大取大，同小取小，大小小大中间找，大大小小无解了。
八

八年级

（上·下两册）
十三角形 八上▼
  • 三边关系：两边之和 > 第三边，两边之差 < 第三边。
  • 内角和：180°。外角 = 不相邻两内角之和。
  • 重要线段：中线（重心）、高（垂心）、角平分线（内心）、中垂线（外心）。
n 边形内角和 = (n−2) × 180°；外角和 = 360°（任何凸多边形！）
判断三条线段能否构成三角形，只需检查"较短两边之和是否大于最长边"。
十一全等三角形 八上▼
判定（5 种）：SSS / SAS / ASA / AAS / HL（仅 Rt△）
SSA（边边角）不能判定全等！两边和其中一边的对角对应相等，三角形可能有两种形状。
十二轴对称 八上▼
  • 垂直平分线：过线段中点且垂直于线段。垂直平分线上的点到线段两端距离相等。
  • 等腰三角形：等边对等角。三线合一（顶角平分线 = 底边中线 = 底边上的高）。
  • 等边三角形：三边相等，三角均 60°。
十三整式乘法与因式分解 八上▼
必背乘法公式
平方差：(a+b)(a−b) = a² − b²
完全平方：(a±b)² = a² ± 2ab + b²
  • 因式分解三步走：一提（公因式）→ 二套（平方差/完全平方/十字相乘）→ 三检查（不能再分）。
x² + 4 在实数范围内不能分解！平方差公式需要减号。因式分解必须分解到底。
十四分式 八上▼
  • 分式有意义：分母 ≠ 0。分式值为 0：分子 = 0 且分母 ≠ 0。
  • 分式方程：必须验根！去分母可能产生增根。
十五二次根式 八下▼
  • 双重非负性：√a 中被开方数 a ≥ 0，且 √a ≥ 0。
  • 性质：√(a²) = |a|（注意取绝对值！）。
√((−3)²) = |−3| = 3，不是 −3！去掉根号时要判断正负。
十六勾股定理 八下▼
勾股定理 · 直角三角形三边关系
勾股定理：a² + b² = c²（c 为斜边）
逆定理：若三边满足 a² + b² = c²，则该三角形是直角三角形。
  • 常用勾股数：(3,4,5)、(6,8,10)、(5,12,13)、(7,24,25)、(8,15,17)
十七平行四边形 八下▼
  • 性质：对边平行且相等，对角相等，邻角互补，对角线互相平分。
  • 判定：对边平行 / 对边相等 / 一组对边平行且相等 / 对角相等 / 对角线互相平分。
  • 矩形：对角线相等。Rt△ 斜边上的中线 = 斜边一半。
  • 菱形：对角线互相垂直且平分对角。面积 = ½ × 对角线乘积。
  • 正方形：兼具矩形和菱形的所有性质。
十八一次函数 八下▼
一次函数 y = kx + b · k 决定倾斜方向和程度 · b 是与 y 轴交点的纵坐标
k > 0：直线上升，y 随 x 增大而增大。k < 0：直线下降。
|k| 越大：直线越陡。b：直线与 y 轴交点 (0, b)。b = 0 时为正比例函数，过原点。
九

九年级

（上·下两册）
十九一元二次方程 九上▼
四种解法：① 直接开平方法 ② 配方法 ③ 公式法 ④ 因式分解法
求根公式：x = [−b ± √(b² − 4ac)] / 2a
韦达定理 · 抛物线与 x 轴交点的横坐标与系数之间的关系
  • 判别式 Δ = b² − 4ac：> 0 两不等实根；= 0 两相等实根；< 0 无实根。
  • 韦达定理：x₁ + x₂ = −b/a，x₁·x₂ = c/a。
二十二次函数 九上▼
  • 定义：y = ax² + bx + c（a ≠ 0）。图像是抛物线。
  • 顶点式：y = a(x − h)² + k，顶点 (h, k)，对称轴 x = h。
y = x² 抛物线 · 对称轴为 y 轴 · 顶点在原点 · 开口向上
|a| 决定开口宽窄 · a 的符号决定开口方向
对称轴：x = −b/(2a)。顶点：(−b/(2a), (4ac−b²)/(4a))。
平移口诀："左加右减，上加下减"——y = a(x−h)² + k 中，h 控制左右平移，k 控制上下平移。
廿一旋转 九上▼
  • 中心对称：绕某点旋转 180° 后与自身重合。P(x, y) 关于原点对称 → P′(−x, −y)。
廿二圆 九上▼
圆的要素 · 圆心角 = 2 × 圆周角（同弧）
垂径定理：垂直于弦的直径平分弦且平分弦所对的两条弧。
圆周角定理：同弧所对圆周角 = ½ 圆心角。直径所对圆周角 = 90°。
切线性质：切线垂直于过切点的半径。切线长定理：圆外一点引两条切线长相等。
圆内接四边形：对角互补（和为 180°）。
  • 弧长 l = nπr/180   扇形面积 S = nπr²/360 = ½lr
廿三概率初步 九上▼
  • P(A) = m/n（m：事件 A 包含的结果数，n：所有等可能结果总数，0 ≤ P ≤ 1）。
  • 列举法：树状图最直观，不重不漏是关键。
廿四反比例函数 九下▼
  • 定义：y = k/x（k ≠ 0），等价于 xy = k。图像是双曲线。
反比例函数 y = k/x（k>0）· 双曲线 · 两渐近线为坐标轴
k > 0：图像在一、三象限，每象限内 y 随 x 增大而减小。
k < 0：图像在二、四象限，每象限内 y 随 x 增大而增大。
k 的几何意义：双曲线上任一点向坐标轴作垂线，所得矩形面积 = |k|。
廿五相似 九下▼
相似三角形 · 对应角相等 · 对应边成比例
判定（4 种）：AA / SAS / SSS / 平行截线。
性质：对应线段比 = k；周长比 = k；面积比 = k²。
廿六锐角三角函数 九下▼
三个特殊角的直角三角形 · 30° / 45° / 60° 的边比关系
定义（在 Rt△ 中，∠C = 90°）：sin A = 对边/斜边   cos A = 邻边/斜边   tan A = 对边/邻边
角sincostan
30°1/2√3/2√3/3
45°√2/2√2/21
60°√3/21/2√3
sin 30° = ½，不是 √3/2！sin 和 cos 在 30° 和 60° 的值刚好互换，容易记混。口诀："三十度，对边是斜边一半"。
  • 互余关系：sin(90°−A) = cos A。 平方关系：sin²A + cos²A = 1。
廿七投影与视图 九下▼
  • 三视图：主视图（正面）、俯视图（上面）、左视图（左面）。口诀：长对正、高平齐、宽相等。
