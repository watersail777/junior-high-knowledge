八

八年级

（上·下两册）
一机械运动 八上▼
  • 长度测量：基本单位 米(m)。1km=10³m，1dm=10⁻¹m，1cm=10⁻²m，1mm=10⁻³m，1μm=10⁻⁶m，1nm=10⁻⁹m。刻度尺读数要估读到分度值下一位。
  • 参照物：判断物体是否运动，需选取参照物。运动和静止是相对的。
  • 速度：v = s / t。1m/s = 3.6km/h。匀速直线运动中 v 不变。
  • 平均速度：总路程 ÷ 总时间，不是速度的平均值。
平均速度 ≠ 速度的平均！如先以 4m/s 走 100m，再以 6m/s 走 100m，平均速度不是 5m/s，而是 200/(25+16.7) ≈ 4.8m/s。
二声现象 八上▼
  • 声音的产生：由物体振动产生。振动停止，发声停止。
  • 传播：需要介质（固体 > 液体 > 气体），真空不能传声。声速 340m/s（15°C 空气）。
  • 声音三要素：音调（频率决定，频率越高音调越高）、响度（振幅决定，振幅越大响度越大）、音色（波形决定，区分不同发声体）。
  • 超声波与次声波：>20000Hz 为超声波（B 超、声呐）；<20Hz 为次声波（地震、海啸监测）。
  • 噪声控制：声源处减弱、传播途中减弱、人耳处减弱。
"声音越大"指的是响度大，不是音调高！女高音的"高"是音调高，不是响度大。两者容易混淆。
三物态变化 八上▼
六种物态变化：熔化（固→液，吸热）、凝固（液→固，放热）、汽化（液→气，吸热）、液化（气→液，放热）、升华（固→气，吸热）、凝华（气→固，放热）。
  • 晶体 vs 非晶体：晶体有固定熔点（冰 0°C、海波 48°C）；非晶体没有（蜡、玻璃、沥青）。
  • 沸腾条件：① 达到沸点 ② 继续吸热。沸点与气压有关，气压越高沸点越高（高压锅原理）。
  • 蒸发：在任何温度下都可发生，只在液体表面。影响因素：温度、表面积、空气流速。
  • 液化方式：降低温度、压缩体积（如液化气罐）。
四光现象 八上▼
光的反射定律 · 入射角 = 反射角
光的反射定律：① 反射光线、入射光线和法线在同一平面内（三线共面）；② 反射光线和入射光线分居法线两侧；③ 反射角 = 入射角。
  • 镜面反射 vs 漫反射：都遵循反射定律。镜面反射从一个方向看亮，漫反射从各个方向都能看到。
  • 平面镜成像：正立、等大、虚像，像与物关于镜面对称。像到镜面的距离 = 物到镜面的距离。
  • 光的折射：光从空气斜射入水/玻璃中，折射角 < 入射角（向法线偏折）。垂直入射时传播方向不变。
光的折射 · 从空气进入水中，折射角 < 入射角
五透镜及其应用 八上▼
凸透镜成像规律：
u > 2f → 倒立缩小实像（照相机）
u = 2f → 倒立等大实像
f < u < 2f → 倒立放大实像（投影仪）
u = f → 不成像
u < f → 正立放大虚像（放大镜）
  • 近视眼：像成在视网膜前，用凹透镜矫正。远视眼：像成在视网膜后，用凸透镜矫正。
六质量与密度 八上▼
密度公式：ρ = m / V。单位：kg/m³ 或 g/cm³。1g/cm³ = 10³ kg/m³。水的密度 ρ水 = 1.0×10³ kg/m³。
  • 质量是物体的属性，不随位置、形状、状态改变。天平使用：左物右码。
密度是物质的性质，与物体的质量和体积无关！不能说"质量越大密度越大"，同种物质密度不变。
七力 八下▼
  • 力的作用效果：改变物体的形状、改变物体的运动状态。力的三要素：大小、方向、作用点。
  • 重力：G = mg（g ≈ 9.8N/kg ≈ 10N/kg）。方向竖直向下，作用点在重心。
  • 弹力：弹簧测力计原理——在弹性限度内，弹簧的伸长量与拉力成正比（胡克定律 F = kx）。
  • 摩擦力：方向与相对运动（趋势）方向相反。增大有益摩擦：增大压力、使接触面粗糙。减小有害摩擦：加润滑油、变滑动为滚动。
八运动和力 八下▼
物体受力分析 · 推力 F、摩擦力 f、重力 G、支持力 N
牛顿第一定律（惯性定律）：一切物体在没有受到力的作用时，总保持静止或匀速直线运动状态。
惯性：一切物体都有保持原来运动状态不变的性质。惯性只与质量有关，质量越大惯性越大。
  • 二力平衡：作用在同一物体上的两个力，大小相等、方向相反、在同一直线上。平衡力作用下物体静止或匀速直线运动。
惯性是性质不是力！不能说"受到惯性"或"惯性力"，只能说"由于惯性"或"具有惯性"。惯性与速度无关！
九压强 八下▼
p = F / S（固体压强） p = ρgh（液体压强）
  • 增大压强：增大压力/减小受力面积（刀刃、图钉尖）。减小压强：减小压力/增大受力面积（书包宽带、履带）。
  • 大气压：托里拆利实验，标准大气压 1.013×10⁵Pa = 760mmHg。海拔越高气压越低（沸点越低）。
  • 流体压强与流速：流速越大压强越小（飞机升力、站台安全线）。
十浮力 八下▼
阿基米德原理：F浮 = G排 = ρ液gV排
浮力只与 ρ液 和 V排 有关，与物体密度、形状、浸没深度无关！
  • 沉浮条件：F浮 > G（上浮）、F浮 = G（悬浮/漂浮）、F浮 < G（下沉）。
物体浸没越深，浮力不变！（V排 已等于 V物）。潜水艇靠改变自身重力实现沉浮，不是改变浮力。
十一功和机械能 八下▼
功：W = Fs（力 × 力的方向上移动的距离，单位 J）
功率：P = W / t（单位 W，1W = 1J/s）
  • 不做功的三种情况：有力无距离（推墙不动）、有距离无力（惯性滑行）、力与距离垂直（提着水桶水平走）。
  • 机械能 = 动能 + 势能（重力势能 + 弹性势能）。动能与质量和速度有关，重力势能与质量和高度有关。
十二简单机械 八下▼
杠杆平衡：F₁L₁ = F₂L₂（动力×动力臂 = 阻力×阻力臂）
机械效率：η = W有用 / W总 × 100%，η 永远 < 1（因为有额外功）
  • 定滑轮：不省力，改变力的方向。动滑轮：省一半力，不改变方向。滑轮组：既省力又可改变方向。

（全一册）
十三内能 九全▼
  • 分子动理论：物质由分子组成；分子在不停地做无规则运动（扩散现象）；分子间有引力和斥力。
  • 内能：物体内部所有分子动能与分子势能的总和。一切物体都有内能。改变方式：做功和热传递。
比热容：Q = cmΔt。c 是物质特性，c水 = 4.2×10³ J/(kg·°C)。
  • 热值：Q = mq（固体/液体）或 Q = Vq（气体）。热机效率 η = W有用 / Q总。
十四电流和电路 九全▼
  • 串联：电流只有一条路径，各用电器互相影响。并联：多条路径，各支路互不影响。
  • 电流：单位 A。串联 I₁ = I₂；并联 I = I₁ + I₂。电流表串连接入，正进负出。
十五电压 电阻 九全▼
  • 电压：单位 V。串联 U = U₁ + U₂；并联 U = U₁ = U₂。电压表并连接入。
  • 电阻：单位 Ω。导体电阻与材料、长度、横截面积、温度有关，与电压和电流无关！
十六欧姆定律 九全▼
欧姆定律：I = U / R。串联总电阻 R = R₁ + R₂；并联总电阻 1/R = 1/R₁ + 1/R₂（并联总电阻小于任一支路电阻）。
欧姆定律 U-I 图像（定值电阻）
R = U/I 不能理解为"电阻与电压成正比"！电阻是导体的性质，不随电压变化。U-I 图像是过原点的一条直线。
十七电功率 九全▼
电功（电能）：W = UIt = Pt（单位 J，1kW·h = 3.6×10⁶J）
电功率：P = UI = W/t = I²R = U²/R（单位 W）
焦耳定律：Q = I²Rt（电流热效应）
额定功率 vs 实际功率——额定功率是灯泡上标的功率（正常工作时的功率），实际功率取决于实际电压。灯泡的亮度由实际功率决定！
十八电与磁 九全▼
  • 奥斯特实验：通电导线周围存在磁场（电生磁）。
  • 安培定则（右手螺旋定则）：右手握住螺线管，四指指向电流方向，拇指所指为 N 极。
  • 电磁感应：闭合电路的一部分导体在磁场中做切割磁感线运动时，产生感应电流（法拉第，磁生电→发电机原理）。
  • 电动机原理：通电导体在磁场中受力（左手定则）。
电动机原理是"电→力"（通电导体在磁场中受力），发电机原理是"磁→电"（电磁感应），二者容易搞混！

物理易错点 Top 10

  1. 惯性不是力——不能说"受到惯性作用"或"惯性力"，只能说"由于惯性"或"具有惯性"。惯性只与质量有关。
  2. 平衡力 vs 相互作用力——平衡力作用在同一物体上；相互作用力作用在两个不同物体上（A对B，B对A）。
  3. 压力 ≠ 重力——只有当物体放在水平面上且无其他力时，压力才等于重力。放在斜面上时压力 < 重力。
  4. 浮力与深度无关——物体完全浸没后，V排不变，浮力不变。潜水艇靠改变自身重力（不是浮力）实现沉浮。
  5. 平均速度 ≠ 速度的平均——平均速度 = 总路程 ÷ 总时间。
  6. 密度是物质特性——不能说"质量越大密度越大"，同种物质密度不变（与质量和体积无关）。
  7. 电阻是导体性质——不能说"电阻与电压成正比"，R=U/I 是定义式不是决定式。
  8. 额定功率 vs 实际功率——额定功率是正常工作时的功率（不变），实际功率随实际电压变化。灯泡亮度由实际功率决定。
  9. 电动机 vs 发电机原理——电动机：通电导体在磁场中受力（电→力）；发电机：电磁感应（磁→电）。不能混淆。
  10. 做功的两个必要条件——有力且在力的方向上有距离。提着水桶在水平路上走，提力不做功（力与距离垂直）。
